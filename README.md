# DeepTrain (Working out made simpler)

### All you need is a set of dumbbells ( Oh come on you are not gonna get fit without spending a few bucks) and a laptop with a cam ofcourse.

## About the Project
- DeepTrain is a machine learning application that helps you exercise at home.
- It will act as your gym trainer and correct your position via looking at your pose.
- DeepTrain is built on tf-poseEstimation library (one of the best and fastest pose - estimation library)
- DeepTrain track your movements and see if your movements are accurate or not.

## What can you do with this application
- Right now the application only supports three exercises. (Shoulder Press, Bicep curl and Tricep PushDown)
- Add a video and see how you did and where you can improve.
- Open camera and get suggestions realtime.
